Topics

  1. Client:

    • A client is a computer or software application that accesses a service made available by a server.

    • Clients initiate requests for services such as web pages, files, or other resources from servers.

  2. Server:

    • A server is a computer or system that provides resources, data, services, or programs to other computers, known as clients, over a network.

    • Servers can serve data and resources via various protocols like HTTP, FTP, SMTP, etc., depending on the type of service they provide.

  3. Domain Name:

    • A domain name is a human-readable address used to identify resources on the internet, such as websites or email servers.

    • It consists of a series of alphanumeric labels separated by dots (e.g., example.com), indicating hierarchy from right to left.

  4. DNS (Domain Name System):

    • DNS is a decentralized naming system for computers, services, or other resources connected to the internet or a private network.

    • It translates domain names into IP addresses necessary for locating and identifying resources.
